Controle de Gastos Domésticos (CGD)

O CGD é uma aplicação desktop desenvolvida para ajudar você a organizar e controlar suas despesas domésticas de forma simples e eficiente.


Versão: 1.0.0
Ano: 2025
Desenvolvedor: Douglas C. Venancio

⭐

Recursos Principais

  • ✅ Cadastro e gestão de despesas
  • ✅ Controle de parcelas e vencimentos
  • ✅ Dashboard com gráficos e relatórios
  • ✅ Filtros avançados e busca
  • ✅ Backup e restauração de dados
  • ✅ Modo claro e escuro
  • ✅ Interface responsiva
💻

Requisitos do Sistema

Resolução:
Mínimo: 1024×768 | Recomendado: 1366×768 ou superior


Navegador:
Chrome 90+, Firefox 88+, Edge 90+, Safari 14+


Armazenamento:
LocalStorage habilitado no navegador
Recomendado: não usar aba anônima/privada
(dados são apagados ao limpar dados do navegador)


Conexão com Internet:
Necessária apenas para carregar o Chart.js
(via CDN do cdnjs.cloudflare.com)
